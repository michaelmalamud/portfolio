GitHub App

pomodorro timer

i use the pomodorro technique in my work flow. i started to wonder why i was using the timer on my phone instead of just using an app on the computer. then I thought, why not just build one myself? you can set your "focus" time up to 60 minutes, and your break time up to 15. anyone who makes use of this productivity method would find utility in this app.

stack: react

GitHub App

flashcard-o-matic

flashcards are probably one of the optimum ways to study, so I wanted to make an app that mimicked them. i think the people using this app (who will likely mostly be students of some kind) will find the quality of their studying to be the same, all while saving on paper and time.

stack: react
